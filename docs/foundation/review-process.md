# Review Process

Every objective passes the following stages.

## Stage 1

Official Objective

- Weight
- Description
- Knowledge Areas
- Utilities

## Stage 2

Technical Review

- Theory
- Commands
- Configuration
- Architecture

## Stage 3

Practice

- Labs (if applicable)

## Stage 4

Exam Preparation

- Exam Pitfalls
- Practice Questions

## Stage 5

Review

Checklist:

☐ All Knowledge Areas covered

☐ All Utilities covered

☐ Labs completed (if applicable)

☐ Questions completed

☐ Objective closed
