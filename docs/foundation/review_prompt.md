# Review Prompt

Perform a final technical review of the current LPIC-3 chapter.

Read first:

- docs/foundation/working-agreement.md
- the official LPI Exam 305 objectives page
- docs/glossary.md
- the current chapter
- the previous chapter (if applicable)

Review task:

- verify objective coverage against the official LPI source
- fix technical inaccuracies first
- add missing exam-relevant topics
- remove duplicate or conflicting explanations
- verify commands, configuration examples and file names
- check glossary coverage for new exam-relevant terms
- keep the smallest reasonable diff
- update `docs/foundation/progress.md` if review status changes

Do not rewrite the chapter unless absolutely necessary.

Act like the final technical reviewer before the chapter is committed.
